# 年次データ更新の手順（年2回）

年次更新は **10月末**と**11月末**の2段階に分かれています。

---

## 【10月末】メイン更新

### ① 総務省サイトで新年度のExcelを探す（Claudeに手伝ってもらう）

Claudeに「今年のデータに更新して」と声をかければ、以下6種類のExcelのURLを一緒に探してくれます：

1. 全市町村・全都道府県の主要財政指標一覧
2. 基金残高等一覧（財政調整基金）
3. 住民基本台帳人口・人口動態
4. 市町村別・都道府県別決算状況調（歳出歳入・実質赤字比率・連結実質赤字比率も同じファイルに含まれる）
5. 目的別歳出内訳
6. 住民基本台帳年齢階級別人口

### ② `scripts/config.json` を書き換える

GitHubアプリで `scripts/config.json` を開き、鉛筆アイコンで編集。
各URLを新しいものに書き換えて、一番上の `fiscal_year_label` と `population_year_label` も新年度に更新してコミット。

```json
{
  "fiscal_year_label": "令和7年度",     ← ここを書き換え
  "population_year_label": "令和8年",   ← ここを書き換え
  "fiscal_indicators": {
    "muni": "https://www.soumu.go.jp/main_content/新しい番号.xlsx",
    ...
  }
}
```

### ③ ワークフローを実行

https://github.com/ray-ray870/mitchie-fiscal/actions/workflows/update-data.yml で「Run workflow」を実行。
安全設計により、検証に成功した場合のみ自動コミットされます。

### ④ index.htmlの年度表記を手動更新

index.html内に直接書かれている「令和6年度」のような年度表記（吹き出しメッセージ、各指標の出典表記、起債許可団体等のバッジ表示など）を、新しい年度に手動で書き換えてコミット。

### ⑤ OGPメタタグ内の自治体数を手動更新

index.html の `<head>` 内にある以下2箇所は、SNS（X・LINEなど）でリンクを共有した際のプレビュー文言に使われる静的なテキストのため、自動更新されません。市区町村の合併・新設などで対応自治体数が変わった場合は、実際の件数に合わせて手動で書き換える。

```html
<meta property="og:description" content="全国1,786自治体の財政をチェック！みっちーが財政健全度を診断します🐧">
<meta name="twitter:description" content="全国1,786自治体の財政をチェック！みっちーが財政健全度を診断します🐧">
```

※ アプリ内の共有ボタン（X/LINEでアプリをおすすめする！）の文言自体は、実際に読み込んだデータ件数から自動生成されるため、こちらの対応は不要。

## 【11月末】確認のみの軽い更新

総務省が毎年11月末頃に公表する「健全化判断比率・資金不足比率の概要（確報）」PDFを確認し、**都道府県が実質赤字比率・連結実質赤字比率で基準超えになっていないか**を照合する。

- 確認先：https://www.soumu.go.jp/main_sosiki/c-zaisei/kouei_kessan.html の「決算の概要（速報）」欄
- 確認する内容：都道府県（47団体）が引き続き「早期健全化基準以上の団体はなし」となっているか
- 該当があった場合のみ、Claudeに伝えて個別対応（ファイルの再取得は基本的に不要）

## 起債許可団体・早期健全化団体・財政再生団体の判定基準

| 指標 | 早期健全化基準 | 財政再生基準 |
|---|---|---|
| 実質赤字比率 | 都道府県3.75% / 市町村11.25〜15% | 都道府県5% / 市町村20% |
| 連結実質赤字比率 | 都道府県8.75% / 市町村16.25〜20% | 都道府県15% / 市町村30% |
| 実質公債費比率 | 25%（共通） | 35%（共通） |
| 将来負担比率 | 都道府県・政令市400% / 市町村350% | 基準なし |
| 起債許可団体 | 実質公債費比率18%以上（共通） | ― |

- 市町村分の実質赤字比率・連結実質赤字比率は「決算状況調（概要）」ファイルに含まれる列（36列目・37列目）から取得（新規ファイル取得は不要）
- 都道府県分の実質赤字比率・連結実質赤字比率は、単体のExcelファイルが総務省から一般公開されていないため、11月末の確報PDFで代替確認する運用としている
- 実質公債費比率・将来負担比率は市町村・都道府県とも既存データからそのまま判定可能

処理スクリプト：`scripts/update_fiscal_data.py`
